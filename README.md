# asteroid-hazard-classification
Data analysis and ML project on NASA Near-Earth Asteroids – Portfolio project

## Project Overview

This project explores NASA’s Near-Earth Asteroids dataset to understand and predict potentially hazardous asteroids (PHAs). 

A Potentially Hazardous Asteroid is defined as an object that comes within 0.05 AU (approximately 7.5 million km) of Earth’s orbit and has a diameter of 140 meters or larger.

Using data analysis and machine learning, this project aims to:
- Explore the characteristics of Near-Earth Asteroids
- Identify patterns related to hazardous asteroids
- Build a classification model that can predict whether an asteroid is potentially hazardous
